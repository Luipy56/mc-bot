# `basic/` — project rule

This folder is intentionally minimal: it is the **baseline template** for new bots. Copy from here when starting a new bot so every project begins from the same simple setup.

**Change this directory only** when you add or adjust **shared defaults** that should apply to future copies (e.g. common scripts, default config, or starter structure). Routine work for a specific bot belongs in that bot’s own project, not here.
