# Roadmap — Minecraft Autonomous Agent

High-level development phases:

1. **Research** — Analyze codebase, Mineflayer APIs, Minecraft progression (Overworld → Nether → End).
2. **Core architecture** — Goal system, task planner, skill system, memory/state, environment perception.
3. **Basic movement and control** — Pathfinding (mineflayer-pathfinder), movement skill, goto goals.
4. **Environment interaction** — Mining, crafting, inventory, chests, block/entity interaction.
5. **Survival systems** — Eating, combat, armor, bed/spawn, night survival.
6. **Progression (Nether, End)** — Portal, blaze rods, ender pearls, stronghold, End dimension, Ender Dragon.
7. **Optimization and autonomy** — Robustness, persistence, task switching, recovery after death.
